# Performance Comparison
Performance comparison between Python, Golang, Rust and C++ using simple Fibonacci code.

## Python

```
python --version
python fibonacci.py
```

## Golang

```
go version
go build fibonacci.go
./fibonacci
```

## Rust

```
rustc --version
cargo build --release
./target/release/fibonacci_project
```

## C++

```
g++ --version
g++ -O3 -o fibonacci fibonacci.cpp
```
